What is Devops?

DevOps is a set of practices, tools, and a cultural philosophy that automate and integrate the processes between software development and IT teams. It is a set of practices which enables teams to devliver high value to users by automating processes.

The term Devops is a combination of two words, Development and Operations

These are two teams involved in Software Devopment. Devops combines both development and operations by making both developers and those in IT operations to work collaboratively throughout the software development life cycle and test to deployment to operations.These teams use practices to automate processes that historically have been manual and slow. Devops makes use of tools and technologies to automate these processes

This process of automation accelerates processes and also increases reliability

The devops symbol shows all the devops stages :

  • Plan
  • Code
  • Build
  • Test
  • Release
  • Deploy
  • Operate
  • Monitor

These stages are arraged in the form of an infinity symbol signifying the continuous activity involved in achieveing a reliable system.

Devops is not a tool but a mindset, which is implemented to achieve a better and more reliable software development life cycle and overall software.
